# Metro Traffic Data Visualization Final Project

Course: CIF444 Data Visualization

This repository contains a Jupyter Notebook project for analyzing the Metro Interstate Traffic Volume dataset from the UCI Machine Learning Repository.

## Dataset

Metro Interstate Traffic Volume  
UCI Dataset URL: https://archive.ics.uci.edu/dataset/492/metro+interstate+traffic+volume

The notebook fetches the dataset dynamically through `ucimlrepo`; no local CSV file is required.

## Technologies

- Python
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- ucimlrepo
- Jupyter Notebook

## Installation

```bash
pip install -r requirements.txt
```

## Running the Notebook

Start Jupyter Notebook:

```bash
jupyter notebook
```

Then open:

```text
notebook/CIF444_Burak_Saltali_Metro_Traffic_Final_Project.ipynb
```

Use **Restart & Run All** to execute the notebook from top to bottom.
